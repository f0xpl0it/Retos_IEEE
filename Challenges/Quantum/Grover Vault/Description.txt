=== GROVER VAULT ===

Category: Quantum / Cryptography
Difficulty: Hard | 500 pts

The quantum vault uses a search oracle to
authenticate the single valid access state among 16 possible ones.

The circuit has been set up and is ready to run.
The system already knows which state is correct.
You do not.

Attachment: grover_vault.py
Dependencies: pip install qiskit qiskit-aer

The state that the universe prefers is the key.
Format: IEEE{XXXX}
