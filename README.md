## 📖 Overview

This repository contains the challenges used during the **IEEE CS Flag Hunters EPN** competition.

The purpose of this repository is to provide access to the challenge resources, descriptions, and supporting files required to reproduce and solve the challenges. Reference flags are **not included** in this repository.

---

## 📂 Repository Structure

```text
.
├── Cryptography/
│   ├── Eight_Commands/
│   ├── Central_Dogam/
│   ├── Prime_Detective/

│   ├── Kyber_Maybe/
│   └── DNA_Vault/
│
├── Quantum/
│   ├── Bell_Fingerprint/
│   ├── QKD_Intercept/
│   ├── Teleport_Tampered/
│   ├── Grover_Vault/
│   ├── Short_Shatter/
│   └── Quantum_Drift_Protocol/
│
└── Forensics/
    └── Redaction_Failure/
```

Each challenge directory contains the resources necessary to reproduce the challenge environment.

Example:

```text
Challenge_Name/
├── description.md
├── challenge/
└── README.md
```
